Instructions:

  1. If you see a big green flag,click it to load the game

  2. Press the space key to start a new game

  3. When you're ready, click inside the game to play!

How to Play:

  • An item of trash will appear on the screen

  • When it reaches the red box, sort into
    either compost or recycle using your arrow keys
    (left for compost, right for recycle)

  • If you sort it in the wrong spot,
    you send trash to the landfill!!

  • Try to get a better score than your friends!
